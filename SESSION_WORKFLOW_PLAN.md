## What Gets Logged

1. **Full transcript** → `website/logs/YYYY-MM-DD_HH-MM.md`
   - The entire conversation in readable Markdown format
   - Generated by `scripts/jsonl-to-markdown.py` from Claude Code's JSONL transcript
   - Preserves the terminal-style format (⏺ for tools, ⎿ for results, > for user)

2. **Summary** → Git commit message
   - Short description of what was accomplished
   - Used as the commit message

## Session Lifecycle - Complete Step-by-Step

### Phase 1: Session Start

**Step 1.1** - You open terminal and run:
```bash
cd ~/a327ex/Anchor
claude
```

**Step 1.2** - SessionStart hook fires, logs timestamp to `.claude/session-timestamps.log`

**Step 1.3** - Claude reads `CLAUDE.md` automatically (Claude Code does this for any CLAUDE.md in working directory)

**Step 1.4** - You state what you want to work on

---

### Phase 2: During Session

**Step 2.1** - Work proceeds normally. No manual logging needed - Claude Code automatically saves the full transcript to `~/.claude/projects/[path]/[session-id].jsonl`

**Step 2.2** - When Claude changes an API or establishes a new pattern, Claude updates:
- `ANCHOR.md` for API/architecture changes
- `CLAUDE.md` for patterns/conventions
- `IMPLEMENTATION_PLAN.md` for completed tasks

---

### Phase 3: Session End

**Step 3.1** - You say "let's wrap up" (or similar)

**Step 3.2** - Claude writes a short summary of what was accomplished

**Step 3.3** - Claude locates the current session's JSONL transcript (path provided by hook or found via most recent file)

**Step 3.4** - Claude runs the converter:
```bash
python3 scripts/jsonl-to-markdown.py ~/.claude/projects/[...]/[session].jsonl website/logs/YYYY-MM-DD_HH-MM.md
```

**Step 3.5** - Claude prepends the summary to the top of the log file (before the transcript)

The log file format becomes:
```markdown
# Session YYYY-MM-DD HH:MM

## Summary
Brief description of what was accomplished.
- Bullet point 1
- Bullet point 2

---

## Transcript

> User message...
⏺ Tool calls...
```

**Step 3.6** - Claude syncs context files:
```bash
cp ANCHOR.md website/context/
cp CLAUDE.md website/context/
cp IMPLEMENTATION_PLAN.md website/context/
```

**Step 3.7** - Claude stages all changes:
```bash
git add -A
```

**Step 3.8** - Claude commits with summary (same text used in log file):
```bash
git commit -m "Summary of session

- Bullet point 1
- Bullet point 2

🤖 Generated with Claude Code"
```

**Step 3.9** - Claude pushes to GitHub:
```bash
git push origin main
```

**Step 3.10** - Claude pushes website to Blot:
```bash
git subtree push --prefix=website blot master
```

**Step 3.11** - Claude confirms:
```
Session complete.
- Transcript saved: website/logs/2025-01-15_14-32.md
- Committed to GitHub: [hash]
- Website pushed to Blot
```

**Step 3.12** - SessionEnd hook fires, logs timestamp

---

### Phase 4: Accidental Closure

**Step 4.1** - Terminal closes unexpectedly

**Step 4.2** - The JSONL transcript exists with everything up to that point (Claude Code saves continuously)

**Step 4.3** - You reopen and resume:
```bash
cd ~/a327ex/Anchor
claude --continue
```

**Step 4.4** - Claude has full context. Ask "where were we?" if needed.

**Step 4.5** - When ready to end, Phase 3 runs normally - transcript includes both parts of the session

---

### Phase 5: Next Session

**Step 5.1** - Start new session:
```bash
cd ~/a327ex/Anchor
claude
```

**Step 5.2** - Claude reads updated `CLAUDE.md`, `ANCHOR.md`, `IMPLEMENTATION_PLAN.md`

**Step 5.3** - Previous session's log is on the website for reference

**Step 5.4** - Cycle repeats

## Hook Configuration (`.claude/settings.json`)

```json
{
  "hooks": {
    "SessionStart": [
      {
        "matcher": "startup|resume",
        "hooks": [
          {
            "type": "command",
            "command": "echo \"[$(date '+%Y-%m-%d %H:%M:%S')] Session started\" >> /home/adn/a327ex/Anchor/.claude/session-timestamps.log"
          }
        ]
      }
    ],
    "SessionEnd": [
      {
        "hooks": [
          {
            "type": "command",
            "command": "echo \"[$(date '+%Y-%m-%d %H:%M:%S')] Session ended\" >> /home/adn/a327ex/Anchor/.claude/session-timestamps.log"
          }
        ]
      }
    ]
  }
}
```
